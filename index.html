Passos para hospedar um projeto no GitHub Pages

  1. Crie um repositorio no GitHub com o mesmo nomedo seu projeto
  2. No terminal, inicialize o repositorio local com git init
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos para git add. e faça o commit com git commit -m "First Commit"
  5. Conecte ao repositorio remoto com: git remote add origin https://github.com//SEU_USUARIO/NOME_DO_REPOSITORIO.git
  6. Envie os arquivos com git push -u origin main
  7. No GitHub, vá em settings > Pages
  8. Em "Branch", selecione main e a pasta /(root). Clique em Save
  9. Após algum segundos, seu site estará disponivel no link exibido em "Saves"
